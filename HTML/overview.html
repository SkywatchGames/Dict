Dict Unity3D Serializable Dictionary.

Overview

Dict is a Unity3D plugin that lets you create dictionaries to stablish associations between keys and values within the Scene Editor.

Your dictionary can be easily created with just a couple lines of code.

The API is very simillar to .NET's builtin generic dictionary.
