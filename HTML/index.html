Dict
Unity3D Serializable Dictionary.

Overview

Dict is a Unity3D plugin that lets you create dictionaries to stablish associations between keys and values within the Scene Editor.

Your dictionary can be easily created with just a couple lines of code.

The API is very simillar to .NET's builtin generic dictionary.

Check out our demo!.

Demo

Guide

1. Import the Dict package into your project.

2. Create a script and declare a public Dict variable.

              public class MyComponent : Monobehaviour {
                  public Dict dictionary;
              }

3. Attach the script to any GameObject.

4. Setup your dictionary. Choose the key and value types and add your entries.

5. Access the stored values by code.

		        	public void MyFunction()
					{
						//get entry count
						int keyCount = dictionary.KeyCount;

						//update or add an entry
						dictionary.Set("main character", GameObject.Find("Character"));

						//remove an entry
						dictionary.Remove("main camera");

						//check if a key is present
						bool hasAnyWeapon = dictionary.Contains("weapon slot 0");

						//iterate over keys and values
						foreach(string key in diciontary.Keys<string>){
							Object obj = dictionary.Get<Object>(key);
							print("Key " + key + " is associated to " + obj);
						}

						//throws a key type exception
						dictionary.Set(2, Camera.main); 
					}

Detailed Description

Dictionary class that maps basic key types to other types in a one-to-many relationship. Although no error will be thrown, repeated keys will trigger incorrect behaviour.
